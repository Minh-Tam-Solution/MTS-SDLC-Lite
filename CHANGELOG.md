# Changelog

## v1.0.0 (2026-02-18) — SDLC 6.0.6

Major overhaul from SDLC 4.8 to SDLC 6.0.6. Complete rewrite of all content.

### Added
- **10-Stage Lifecycle** (replacing 4-stage model)
- **4-Tier Classification** (LITE / STANDARD / PROFESSIONAL / ENTERPRISE)
- **SE4H/SE4A Governance Model** (Human Coach + Agent Executor)
- **Quality Gates** (G0.1 through G4 + G-Sprint)
- **8 SDLC Roles** (researcher, pm, pjm, architect, coder, reviewer, tester, devops)
- **4 Team Archetypes** (planning, dev, qa, fullstack)
- **Multi-Provider Strategy** (model assignment per role)
- **Design Thinking Foundation** (5 phases mapped to stages)
- **LITE Quickstart** (30-minute setup guide)
- **Crisis-to-Pattern Playbook** (turn failures into reusable assets)
- **Project Config Template** (.sdlc-config.json for LITE tier)
- **Gate Checklist Template** (pass/fail criteria per gate)
- **TinySDLC Reference** (case study of 6.0.6 LITE implementation)
- **Lite vs Enterprise Comparison**
- **Glossary** and **Version History**

### Changed
- Language: Vietnamese-primary → English-primary (international community)
- Framework base: SDLC 4.8 → SDLC 6.0.6
- LITE tier: 4 stages → 5 stages (added 03-INTEGRATE)
- File naming: PascalCase → kebab-case (per SDLC 6.0.6 standards)
- Audience: Event-specific → General OSS community
- All playbooks and templates rewritten for 6.0.6

### Removed
- `05-Presentation-Materials/` (event-specific content)
- All event-specific references
- Event-specific QR codes and booking links
